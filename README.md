# HEIC to PNG Converter

A free, unlimited, privacy-focused HEIC to PNG converter web application built with FastAPI and vanilla JavaScript. No sign-ups, no paywalls, no file limits.

## Features

- 🚀 **Instant Conversion** - Convert HEIC files to PNG in seconds
- 📦 **Batch Processing** - Upload and convert multiple files at once
- 🔒 **Privacy First** - Files processed server-side and immediately deleted
- 📱 **Mobile Friendly** - Works perfectly on iPhone for AirDrop transfers
- ⚡ **Real-time Progress** - Live conversion progress for each file
- 📊 **Usage Stats** - See total conversions in real-time
- 🎨 **Modern UI** - Clean, intuitive interface with smooth animations

## Tech Stack

- **Backend:** Python 3.11+, FastAPI, MongoDB (Motor async driver)
- **Image Processing:** pillow-heif + Pillow
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Database:** MongoDB for anonymous usage statistics

## Prerequisites

- Python 3.11 or higher
- MongoDB 4.4 or higher
- pip (Python package manager)
